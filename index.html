Welcome to theastrologer-api¶

theastrologer-api is an un-official REST API that exposes readonly GET endpoints to see your horoscopes from theastrologer.com website

The api is written in Flask framework using theastrologer python library. It also allows you to send today’s horoscope as SMS using Twilio API

The code is open source, available on github.
